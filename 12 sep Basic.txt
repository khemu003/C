-> what is c ?

-> what is structural programming language ?
	-> It means that it have a particular structure and fixed structure to make a code
	-> like in c, first we write header file than main function.

-> what is procedural programming language ?
	-> It follow the process or it execute code step by step
	-> like making tea
		-> first we boil the water
		-> than add milk
		-> than sugar and tea power

-> what is platform depend ?
	-> It means, we write a code in one device to another device but code is not easy to run on another device.
	-> like some apk files
